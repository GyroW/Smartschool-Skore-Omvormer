# Smartschool-Skore-Omvormer
Zet skore punten om naar een handig af te drukken formaat om in te dienen.
# Requirements
* Python
* twisted
```code-block:: bash
    apt-get install python-twisted
```
* lxml
```code-block:: bash
    pip install lxml
```
of 
```code-block:: bash
    sudo pip install lxml
```
